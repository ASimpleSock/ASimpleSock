Running AI Object Detection on CCTV Cameras.

Okay so the title is a lie, I'm only running it on a single camera because my Raspberry Pi has reached its limits.

But look at how much more info we get now! That Front Door Motion Sensor I just installed is now redundant.

Plus it keeps a track of every person who came inside! And yes only inside, a simple mask prevents it from detecting people on the road.

I should mention all this is run locally 24x7 using Frigate.

And since it integrates completely with Home Assistant, the possibilities really are limitless.
